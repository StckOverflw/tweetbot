# Tweet Bot
A Twitter Bot that Tweets a custom Text or a random Text from a list after a specific time

<details>
<summary>.env</summary>

```properties
TWITTER_API_KEY=
TWITTER_API_SECRET=
TWITTER_ACCESS_TOKEN=
TWITTER_ACCESS_SECRET=

KRONTAB_SCHEDULE="/30 * * * *"
TWEET_TEXT="Tweet Option 1|Tweet Option 2|Tweet Option ..."
```

</details>
